About

eXtensible Admin is a fully featured admin framework built on the EventScripts 2.x plugin and written in Python. XA comes with numerous core modules to manage administration, commands, settings, languages, menus, Auth and more. XA is fully customisable with a low barrier to entry: custom modules can be created and integrated in minutes.
